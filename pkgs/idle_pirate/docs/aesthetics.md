# Aesthetics and Icon Style

This document defines the visual style for the Idle Pirate game assets.

## Icon Style
-   **Description**: Stylized, vibrant cartoon style fitting a pirate theme but with rich aesthetics.
-   **Key Features**:
    *   Smooth gradients.
    *   Dynamic lighting and glowing effects.
    *   Dark mode compatibility (isolated on a dark background).
    *   Vibrant colors that pop against a dark theme.
-   **Size**: Approximately 200x200 pixels.
